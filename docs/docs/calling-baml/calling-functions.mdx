For a BAML function defined like this:

```rust
enum MessageCategory {
    Refund
    CancelOrder
    TechnicalSupport
    AccountIssue
    Question
}

function ClassifyMessage(...) -> MessageCategory {
  client GPT4Turbo
  prompt #"..."#
}
```

<CodeGroup>

```python Python
import asyncio

from baml_client import b
from baml_client.types import Category

async def main():
    category = await b.ClassifyMessage("I want to cancel my order")
    assert category == MessageCategory.CancelOrder

if __name__ == '__main__':
    asyncio.run(main())
```
