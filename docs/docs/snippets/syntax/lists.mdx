If you have to declare a list in a .baml file you can use this syntax:
```rust baml
{
  key1 [value1, value2, value3],
  key2 [
    value1,
    value2,
    value3
  ]
  key3 [
    {
      key1 value1,
      key2 value2
    }
    {
      key1 value1,
      key2 value2
    }
  ]
}
```
<Tip>The commas are optional if doing a multiline list.</Tip>
