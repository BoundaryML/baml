https://huggingface.co/ - Community for viewing and serving open-source models

HuggingFace supports the OpenAI client, allowing you to use the [openai](/docs/snippets/clients/providers/openai) provider with an overriden `base_url`

See https://huggingface.co/docs/inference-endpoints/index for more information on their Inference Endpoints.

```baml BAML
client<llm> MyClient {
  provider openai
  options {
    base_url "https://api-inference.huggingface.co/v1"
    api_key env.HUGGINGFACE_API_KEY
  }
}
```
