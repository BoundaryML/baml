
https://openrouter.ai - A unified interface for LLMs

OpenRouter supports the OpenAI client, allowing you to use the [openai](/docs/snippets/clients/providers/openai) provider with an overrided `base_url`


```baml BAML
client<llm> MyClient {
  provider openai
  options {
    base_url "https://openrouter.ai/api/v1"
    api_key env.OPENROUTER_API_KEY
    model "openai/gpt-3.5-turbo"
    headers {
      "HTTP-Referer" "YOUR-SITE-URL" // Optional
      "X-Title" "YOUR-TITLE" // Optional
    }
  }
}
```
